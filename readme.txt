[en]

If you want to change any settings:
You need the src.json file
Below I will explain which line exactly means what.

pallette - preset color schemes (I advise you not to touch)
display - display settings (about them below)
gamelimit - the limit of passing games (if the limit is crossed, the game is considered completed)

About display:
	size - window size (do not touch!!!)
	pix - pixel size (how big will it be)
	fps - screen refresh rate
	pallette - the selected color scheme (selection from the pallette list)
	border - the presence of edging in pixels
	value - music volume
	lang - text language

[ru]

Если хочешь поменять какие-нибудь настройки:
Тебе нужен файл src.json
Ниже обьясню - какая строчка что конкретно означает.

pallette - заданные цветовые схемы (советую не трогать)
display - настройки отображения (о них ниже)
gamelimit - предел прохождения игр (если пересечен лимит - игра считается пройденной)

Про display:
	size - размер окна (не трогать!!!)
	pix - размер пикселя (насколько будет большим)
	fps - скорость обновления экрана
	pallette - выбранная цветовая схема (выбор из списка pallette)
	border - наличие окантовки у пикселей
	valume - громкость музыки
	lang - язык текста
